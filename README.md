# Site

This repository contains a static website with HTML, CSS, JavaScript, and TypeScript source files.

## Structure

- `index.html`, `mentoria.html`, `metodologia.html`, `page.html`, `para-empresas.html`, `para-voce.html`, `quiz-inicio.html`, `quiz.html`, `resultado.html`
- `assets/` for images
- `css/styles.css`
- `js/` for JavaScript files
- `src/` for TypeScript/React source files

## Publish

This repo can be hosted on GitHub Pages or any static site host. To deploy using GitHub Pages, enable Pages in repository settings and point to the `main` branch.
